Web Digital Footprints

Privacy Policy

Introduction

Web Digital Footprints is a Master's thesis' project that aims to create a browser extension that analyses users' web browsing habits. The goal is to raise awareness about what kind of information is left behind while browsing, and how these informations can be used by third parties to infer personal informations. All the informations gathered, computed and displayed can be used to both help us improve our program, and also be used anonymously for our academic research.

By installing the extension, you agree that developer handles your data in the way this document discloses it. All the data that the extension collects is sent to our servers to be centralized and compared to other browser's data. This is used to give you enhanced stats about your browsing habits.

Data collection

When you install the extension, your browser is assigned a randomly generated token that is used to identify this browser. That token is transmitted with all future communications with our server. It is the only piece of information that is locally saved on your browser. While installed, the extension collects and sends the following informations :

  • Accessed URLs : Each time a an URL is visited, the extension registers the following : browser's extension token, current time, host and path of the URL (we voluntarily avoid registering the query part of the URL)
  • URLs watch time : We count the time you spend having a tab selected. We register the following : browser's extension token, time spend on each URL which is composed of only host and path of the URL (we again voluntarily avoid registering the query part of the URL)
  • Requests sent : We register metadata about the requests your browser sends when browsing. Each time a request is sent, we register the following : browser's extension token, origin URL (containing again only host and path), contacted URL (containing again only host and path), and total size of the request's body (not the answer).
All the described data are not sent directly : They are gathered and sent every 30 seconds to our servers. Our servers are located in Switzerland.

Data usage

We aggregate and launch some computations on the data we recieve in order to give back insightful visualisations on your own data. These are mostly summing up the total amount of visits on URLs on the total watched time of URLs. We also use the URLs visited to try to indicate you what our algorithms think your interests center are. We do this by downloading ourselves a public version of the URLs you visit and analyse the content of the page.

Data sharing

We do not share the data we collect with other third-parties.
